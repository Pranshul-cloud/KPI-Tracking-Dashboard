A dynamic and interactive KPI Tracking Dashboard built in Power BI to monitor monthly business performance against previous year benchmarks. The dashboard helps stakeholders quickly track growth, revenue trends, customer behavior, and operational performance through real-time KPI monitoring.

---

## 🚀 Features

- Dynamic KPI tracking
- Year-over-Year comparison
- Interactive slicers (Location & Month)
- Live dashboard refresh capability
- Clean and executive-level UI design
- Growth/decline indicators with visual storytelling

---

## 📌 KPIs Included

- Gross Profit
- Net Revenue
- Total Discount
- Total Contracts
- Service Revenue
- Avg Contract Value
- Avg Renewal Rate
- Total Footfall
- Total Transactions
- Avg Basket Value
- Avg Walk-In %
